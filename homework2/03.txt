Думаю, что для реализации игры Blackjack нам потребуется 3 класса:

1) class Player - содержит основную информацию о конкретном игроке

В конструкторе класса зададим 3 поля:
- string Name: Имя игрока
- array Cards: Карты игрока (пустой массив)
- int Bet: Ставка игрока

Также в классе будет 2 метода:

void playerMove(array &CardsDesk): ход игрока
В этом методе мы будем предлагать игроку взять новую карту, до тех пор пока он не откажется.
Если игрок берет карту, то она добавляется к нему в поле Cards, и убирается из общей
колоды CardsDesk, которую мы передадим в этот метод по ссылке.

void setCards(array Cards): получение первых двух карт при раздаче
метод будет заполнять поле Cards этого экземпляра класса.


2) class Dealer наследуется от Player - в нем мы добавим логику работы дилера (крупье)

void playerMove(&CardsDesk): переопределим метод из родительского класса,
вместо того, чтобы предлагать игроку взять карту, здесь мы реализуем логику дилера.


3) class Game - непосредственно сама игра

В конструкторе класса зададим 2 поля:
- array CardsDesk: Перемешанная колода карт
- array Players: Игроки (массив из экземпляров Player)

Также в классе будет 3 метода:

void cardDraw(): Раздача карт
метод будет брать по 2 карты из CardsDesk и передавать их в метод setCards каждого игрока и дилера.

void getResult(): Подведение итогов игры
В этом методе мы проверим карты всех игроков и посчитаем кто сколько выиграл или проиграл.

void play(): Игровой процесс
вызовем метод cardDraw(), затем будем вызывать метод playerMove() для каждого игрока и дилера,
в конце вызовем метод getResult()
